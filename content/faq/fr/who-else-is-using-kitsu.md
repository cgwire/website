## Conçu pour tout le studio, approuvé à tous les niveaux

Kitsu est adopté par les personnes qui font le travail :

- Les producteurs l'utilisent pour suivre les budgets, prévoir les coûts et générer des rapports prêts pour les clients
- Les directeurs et superviseurs s'appuient sur lui pour les revues de playlist, les outils d'annotation et la comparaison de versions
- Les artistes tiennent leurs listes de tâches à jour et soumettent des aperçus sans quitter leur outil DCC
- Les coordinateurs de production gèrent la répartition des tâches, la planification et l'allocation des ressources dans une seule vue

Les choses cessent de passer entre les mailles du filet quand toute l'équipe est dans le même système.

## Open Source. Vérifié de manière indépendante.

Contrairement aux outils propriétaires en boîte noire, Kitsu est open source :

- Vous pouvez inspecter le code avant de vous engager
- Des centaines de studios et de développeurs ont contribué à la plateforme et l'ont mise à l'épreuve
- Pas d'enfermement propriétaire. Vos données vous appartiennent

La transparence est notre façon de fonctionner.
